How to use Ape:

Download Python and while installing, select "Add Python to PATH"

Download Ape

Download Forge 1.8.9

Run the game with Minecraft Forge 1.8.9 (Do not use too many mods, use only basic mods like Optifine etc, and don't use mods that contain "Essential" in them, which is ususally seen in a lot of Sk1er mods) Join a singleplayer world and pause the game.

Extract the downloaded ape file by right clicking it (make sure to extract it into a seperate folder)

Go into the extracted folder and open requirements.txt and then delete the version number after "pandas"

Press CTRL + S to save the file.

Now, run requirements install.bat.

If it doesn't work, go to the adress bar where it shows the folder name on the top, and double click it and type "cmd" and press enter, and in the window that pops up, type pip install -r requirements.txt

Now wait for a bit until it's done.When it's done, open "server.py" from the folder or type in command prompt py server.py or python server.py Wait for it to show "Listening on localhost" at the end.

Once it's done, open the Vape_V4 or Vape Lite folder (which one you want) and drag Vape_V4.exe onto Kangaroo Patcher.exe and click on "Yes" when it asks for permission. Now, just wait for it to inject into Minecraft Forge, and it should work!
